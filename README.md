**Fivem Script Converter**


*Usage*

```js
const converter = require('fivem-script-converter');

// to convert a qbus script to esx
converter.qbustoesx();
// ===============================


// to convert a esx script to qbus
converter.esxtoqbus();
// ===============================
```
